# 創作計畫書 Da Capo
## 一、主題背景介紹
### 故事宇宙時空及創作構想
    
一個大學生撰寫完的程式專題，不慎留下了許多漏洞，導致在繳交完成之後，此程式依舊在電腦的背景裡不知不覺的運行著，而因為作者不關機的小習慣，讓這個程式的世界得以自由的繼續發展下去。  
  
這個程式模擬出一個異世界「歐克里斯大陸」的運行，初期設定如地名、物理法則、基本粒子的確切資訊等均由學生一手完成，並且撰述了正式運行前兩百年的歷史。這兩百年的歷史僅僅只是作為前情提要的存在，學生並未詳細設定使得這段時間bug百出，世界中因此頻頻出現異象。一直到程式正式運行，這個世界才真正穩定下來。學生將該年作為曆法的起始，也就是阿雷格爾元年。阿雷格爾，取自音樂速度標示Allegro(快板)，也預示著這個世界的時間流速快於學生所在的現實世界。除了曆法的命名以外，世界中事物的命名也幾乎都與音樂詞語有關連，例如世界中的「歐克里斯」大陸與「特爾拉」海取自orchestra(管弦樂團)；其中的城邦名稱「麥索」、「弗特」與「皮埃諾」取自音樂強弱符號mazzo-、forte與piano；魔物存在的世界「康切爾托」取自concerto(協奏曲，亦有競爭的涵義)；而基本粒子「諾特」取自構成樂譜的重要元素——音符(note)，儘管這些詞語都有各自的涵義，但在語言發展脈絡沒有被設定完全的歐克里斯大陸居民眼中，這些詞語就像是憑空冒出來一般，毫無軌跡可循。其餘還有很多學生設定上的漏洞，連同兩百年前沒有設定的歷史空白一起被稱為「阿雷格爾謎團」。而在這個作者僅撰述了200多年、再加上世界自動運行的200年間，經過了兩次魔王大混戰時期後，終於邁入一個和平的時代，人們終於有一些閒暇與餘裕而開始追溯歷史、研究這個世界的形成與運行的法則。  

極為讓研究者熱衷的難題便是攻克「阿雷格爾謎團」，經由多代的以考古學家為主的各領域研究者不懈的分析與實地探勘，進行推算，終於描繪出了整個世界史，但總有一個神秘而無法觸及的年代，他們認為那就是這個世界的起源，其中一名研究者萊夏．菲爾德首先意識到這個問題，提出了「阿雷格爾謎團」並將這個年代命名為「時間斷崖」，往後的學者也繼續沿用這個稱呼，至今還沒有人能回到時間斷崖並留下關於它的研究紀錄，只要是有關時間斷崖的一切都是一去不復返，但是這並不影響研究者們的熱情，他們現今依舊想找出真正的答案。最後，他們將目光放到這個世界「科技」的代名詞——魔法身上。「既然無法透過後世的線索推算出答案，那就直接回到當時的時間點親眼看看吧!」懷著這樣簡單直接的想法，以研究現存魔法、創造新魔法為業的術者與術學家接著注意到了曾經留下的研究紀錄，那是一種能夠回到過去的魔法，也就是時間回溯魔法「Da Capo」。這項研究曾因為違反科技倫理遭到政府「長老會」阻撓，如今長老會經過多次成員輪替，變的開明許多。在現今長老會成員的支持下，時間回溯魔法的研究重啟，並且取得了極大的進展。在與長老會簽訂「往昔之物不干涉臨時條款」確保不違反未來走向後，得到政府許可的第一次時間旅行即將展開，其目的為回到時間斷崖以見證世界的起源。  

玩家所扮演的主角因普斯特出生於和平年代，對解開「阿雷格爾謎團」抱持極大的熱忱。此時恰逢時間回溯魔法研究重啟，他依靠不俗的魔法造詣加入了最興盛的研究組織——魔法學會，成為「Da Capo研究組」的成員。隨著時間回溯魔法研發完成，他自告奮勇接下了時間旅行的任務，由於時間回溯魔法的技術發展尚未達到能夠直接回到阿雷格爾前200年的時間斷崖的地步，主角將通過數個中繼時間點，才能夠回溯至時間斷崖，由於往昔之物不干涉臨時條款的約束，停留於中轉時間點的期間不得對歷史進行干涉。這是十分重要的選擇節點，一旦選擇錯誤，違反了未來走向，整個世界將會崩潰並進入死亡結局。  

在停留於中繼時間點的過程中，主角會發現時間斷崖或許並不僅僅是一個無法推算的空白歷史，而這個世界似乎藏著更大的秘密。我們將引導玩家去發現魔物的來源，在時間回溯下來到魔物討伐熱潮開始前，在第一任魔王的引導下玩家將去到另一個世界---「康切爾托」，發現康切爾托主要的住民就是在原世界令人恐懼的魔物，在玩家的視角下會認為魔物是從康切爾托流過來的生物，但事實上康切爾托其實是隔壁資料夾的因為同樣原因在背景執行的程式，因為作者留下的漏洞導致兩個程式間用同一塊 shared memory，所以在讀寫的過程中，主角所在的歐克里斯大陸存取到康切爾托的 data，而導致有生物的流動。而因為世界運作的方式不同所以在流過去的生物被當地的住民視為魔物。  

康切爾托因為是作者更早期的專題作業，所以漏洞層出不窮，我們會讓玩家一一去體驗這些異象，讓玩家逐漸解開他所在的這兩個世界真正的樣貌。比如說 pointer 處理錯誤導致對話噴亂碼；對記憶體操作不當導致對應程式中止執行，使得主角正在觸碰的物體無預警地消失...等等，並讓玩家注意到如果人口太多(其實是記憶體不足)，會導致世界不定期出錯，然後陰錯陽差地讓他知道如何去操作把生物移動到別的世界，當玩家回到原本的世界歐克里斯大陸，為了能讓它保持正常運行而開始清理人口，透過特殊手段將生物移去別的地方<自行name>，並不會致使其生命的消亡，但被當時的人認為他在進行大屠殺而稱他為第二個魔王，但後來發現所作費時費力又不討好且治標不治本，便想著如果去到時間的盡頭，或許能搞懂這個世界真正的運作，從而從根本上改變這樣的現狀。  

所以他時間回溯到歷史上最不穩定得時期，也就是元年前一天，他推測這時期的不穩定是世界運行的不穩定所造成的，因此想來到這個時期去看看是否能利用這些不穩定因素回到時間斷崖，最後終於讓他發現時間斷崖的本質就是一片不被定義的區塊，被世界賦予定義的生命是沒辦法回到時間斷崖裡，主角必須去除自身的定義後才能成功回到時間斷崖。  

時間斷崖的真相：時間斷崖並不是無法用時間回溯到達，而是因為儘管進入那個年代，就會一直被困在世界起始點的那一天不斷循環，因為目前沒有任何人成功在時間斷崖內把關於起源的研究紀錄脫離循環留傳下來，所以人們才會認為它神祕而無法觸及。但實際上它是個 while loop，我們會讓玩家體驗到幾輪的對話之後，就會開始重複，但在程式的邏輯上它並不是絕對無法脫離的，所以玩家要去尋找出離開循環的方法。可以通過達成 break 的條件，或是造成會讓程序不穩定、停擺的因素都可以脫離出去，並通向不同的結局。

最後的最後，其中有一個大方向的結局是，玩家來到了電腦外面發現，原來他所熟知的世界，只不過是電腦裡運行的一個程序而已，依據先前不同的選擇分支會達到最後的結局。從上帝視角來看，本作品會有兩個面向，「歐克里斯大陸」和我們所在的現實世界，作品本身會全部以因普斯特的視角出發，慢慢引導玩家去發現現實世界的一些痕跡，最後打破第四面牆來到現實世界並了解原來一切到底是怎麼一回事，先前經歷過的故事及因普斯特拼命去破解的世界的謎題原來不過是程式運作出來的結果，讓玩家帶入箱庭中的人物並最後告訴全部都是虛假的，對於這個事實該如何去面對，則會分為不同的四個結局。

### 類別說明以及其後設資料欄位設定
#### 職業
這個世界的居民的日常性勞動，目的為獲取勞動所得(即薪資)以維持生活所需，亦象徵在社會上的地位和名譽等等。
- 工作型態  
該職業過去與現今的工作內容。
- 專業制度  
得到該職業之從業許可所要求的門檻，包含執照與資格考試。
- 發展歷史  
該職業從古至今的興衰發展。
- 辭源  
該職業稱呼之來源。
- 種類  
該職業其下再依據細部的工作內容、完成工作的方式，進行更細緻的分類。
- 職階  
依照專業能力強弱進行劃分，同一職業之間的位階分別，通常會透過職階差異賦予從業者不同的權限。
#### 學術研究
這個世界的居民透過研究行為所得出的世界運行規則、尚未完全分析透徹的議題，以及重要物質。
- 發展歷史  
該研究的起因，以及研究過程中的重要時間點，包含取得的巨大進展或是受外力影響而停止的確切年分。
- 組成  
若研究內容為物質，此欄位包含該物質更為詳細的構造介紹，或是物質總稱下更細緻的種類劃分。
- 性質  
若研究內容為物質，此欄位包含該物質所展現出來的特性。
- 應用  
研究成果實際運用、或是預計將運用的領域。
- 解  
對尚未完全分析透徹的理論與問題，各學派產生的不同說法。
#### 歷史事件
世界過去的發生事件和行動，以及對這些事件行為有系統的記錄、詮釋，供人們理解過去。
- 日期  
事件開始與結束的年代。
- 地區  
整起事件所涉及的全部地區。
- 起因  
事件發生的導火線與其背景。
- 過程  
事件所經歷的階段。
- 結果  
事件結束的狀態。
- 影響  
事件對後續未來造成的影響。
#### 組織
由政府或民間成立，包含2人以上成員、曾經或至今進行活動的團體。
- 歷史  
該組織從古至今的興衰發展。
- 領導人  
可能負責整個組織的運行，代表著組織的人物。
- 創立日期  
組織成立的時間，或是經歷改組後重新運行的時間。
- 位址  
組織總部坐落的地方，會是組織成員主要進行活動的地方。
#### 人物
記錄人的相關詳細訊息、特性，以了解此人的所作所為。
+ 性別  
人物的生理性別，可能為男性、女性，或是該種族特有的性別
+ 個性  
人物性格、特色。
+ 職業  
人物所從事的職業。
+ 生卒年  
出生與死亡的時間。
+ 存活年代  
若人物曾經歷時空回溯，此條目將記錄他所有存在過的時間區段及順序。
+ 活躍地區  
人物主要活動的範圍。
+ 主要事蹟  
人物發表過的言論或著作，或是參與過的重大事件與貢獻。
#### 地區
包含行政區域如城邦，或是地理學意義上的大範圍區域如陸地、海洋。
- 位置  
地區所在的絕對位置。
- 所屬人/組織  
地區內主要的勢力、抑或是統治者。
- 面積  
所佔的區域面積。
- 語言  
在此地區的住民主要使用的語言，可能不只一種。
#### 生物
世界上的物種分類與詳細介紹。
- 規模  
在世界裡存在的族群大小。
- 起源  
生物出現的起源，包括起源地、出現時間。
- 體型  
生物的外型特徵。
- 習性  
生物的生活習慣、行動的特性。
- 分布  
生物的棲息地範圍。
- 文化  
生物經演變發展出來的群體行為。
#### 法律規範
由規則組成的體系，由官方強制力保證實施，規範個人行為。
- 發展歷史  
該法律的設立原因，以及設立過程中的重要時間點。
- 主題  
該法律的詳細內容，作用對象與特例等。
#### 列表
用來歸納和列舉一類相關主題的頁面。
- 地區  
列表內事物所在位置。

## 二、多線敘事設計
### 表現方式及媒材設計
+ 本多線敘事故事會使用Ren'Py進行製作，Ren'Py是以python進行創作視覺小說的程式。  
+ 故事主體以文字呈現為主。
+ 會在背景加上符合故事進程的場景圖片，還有故事人物的插圖，配合場景的轉場及人物圖片的滑進滑出可以做出特殊演出的效果。圖片的素材會先以網路上提供的免費素材為主。  
+ 配合敘事插入適當的bgm及特殊音效，在劇情平緩時放輕柔的音樂，劇情緊湊放緊張的音樂，嘗試加入魔法的音效讓玩家更有體驗。音樂的素材會先以網路上提供的免費素材為主。  
### 故事藍圖
#### 主要結局
1. 出來現實世界滅世刪檔案
2. 出來現實世界救世：叫作者修bug
3. 出到os層在檔案外面修復世界(保留結局)
4. 留在原本世界滅世：作者有一天心血來潮打開之前的專題，發現檔案毀損

### 互動機制說明
#### 選項
故事進行中，部分文本下方會出現可供讀者選擇的選項，選擇後將會立即影響劇情走向，決定本次閱讀的分支圖路線。

#### 正義值
正義值初始值設為0，故事中隨著劇情進展呈現主角的內心活動供讀者選擇，其中一個選項較為正面、積極，使正義值增加；另一個較為負面消極，使正義值減少。<br>
終章中主角得知世界的真相後，會直接出現「是否毀滅這個虛假的世界」等類似內容的選項，選擇的結果將以較大的加權與之前累計的正義值進行結算，若正義值為正值，會進入救世結局，反之進入滅世結局。

#### 獲取線索
主要出現於《康切爾托》章節，文本中的場景敘述、與故事角色的對話內容會設置按鈕，點選後將會對按鈕所指代的事物進行觀察或更動以獲取線索，若觸發設置於對話中的按鈕則視為主角向對方提起相關話題。所有推動劇情進展的必要線索全數收集完畢後，故事才會繼續進展下去。  
如果在得到前往現實的線索之前就將必要線索收集完成，劇情會繼續進行，讀者將錯過獲取前往現實的線索的機會。  

#### 脫離時間斷崖的方法
在章節《時間斷崖》中，即使按下「下一頁」按鈕，也只會重複顯示同一個文本，表示主角正在重複經歷同一天。一旦輪迴次數過多，主角將會永遠迷失在時間斷崖中。<br>
與「獲取線索」玩法相似，文本中的部分文字會做成按鈕，點選後視為主角對按鈕所指代的事物進行觀察或是更動，點選到特定按鈕(如「離開」)就能脫離時間斷崖。
